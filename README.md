# Clinic Management System

## Project Overview
This project is a Python-based Clinic Management System built using object-oriented programming (OOP). It allows users to manage patient records, track visit history, and analyze health data such as blood pressure and BMI.

## Features
- Add new patients
- Record patient visits with blood pressure and weight
- Automatically calculate BMI
- Provide health advice based on vitals
- View all patients
- View detailed visit history with dates and advice

## Code Structure

### Patient Class
- Stores patient information (name, DOB, height)
- Maintains visit history
- Calculates BMI
- Provides blood pressure and weight advice

### ClinicSystem Class
- Manages all patient records
- Handles user interaction via menu
- Allows adding patients and visits
- Displays patient and visit history

### Main Function
- Entry point of the program
- Runs the CLI menu system

## How to Run
1. Make sure Python is installed
2. Run the file:# clinic-management-system
This project emphasizes Python object-oriented programming by modeling patients and clinic as classes. The system stores and manages patient details including personal info and health metrics with historical tracking.

## Example Usage
- Add a patient
- Add a visit (enter BP and weight)
- View visit history with health advice

## Technologies Used
- Python
- Object-Oriented Programming (OOP)
